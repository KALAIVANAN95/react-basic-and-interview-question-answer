  };
}

6.Difference between == and ===?
i)== (loose equality): compares values after type coercion (type is not strictly checked)
ii)=== (strict equality): compares both value and type (no type coercion)
== checks value only.
=== checks both value and datatype.
I prefer === to avoid unexpected type coercion.
