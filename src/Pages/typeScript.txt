Since TypeScript is a superset of JavaScript, I am comfortable understanding the syntax and can work with existing TypeScript codebases.

1.What is TypeScript?
TypeScript is a superset of JavaScript that adds static typing to JavaScript.

2.Why use TypeScript?
TypeScript helps catch errors during development and improves code maintainability.

3.Interface
Interface is used to define the structure of an object.

4.Type 
Type is used to define custom data types and object structures.

5.Difference between Type and Interface?
Both are used to define object structures. Interface is mainly used for object definitions, 
while Type can define objects, unions, and other custom types.

6.Union Type.
Union Type allows a variable to have multiple types.

7.Optional Property?
The ? symbol indicates an optional property.

8.Any?
Any type can store any value and disables type checking.
